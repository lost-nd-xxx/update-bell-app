# アイコン仕様案

## アプリ名（日本語）
- おしらせベル
- 文字入りアイコンでは省略して「おしベル」と表記
  - 本来の意味は「お知らせ＋ベル」だが、「推し＋ベル」や「教える」とイメージが重なるのが好印象。

## メインモチーフ
- 中央からやや右下寄りにベル本体
- ベル左上部からWi-Fi電波の波紋（放射状半円）
- 波紋は2本、画像サイズ極小のときは省略。

## スタイル方向
- ミニマルフラット・現代的UI風（マテリアルデザインなど）
- 極小サイズ（16×16）にも対応できる単純化したシルエット
- アダプティブアイコンやfaviconにも流用可能

## 配色
- 背景色は既存ローディング画面の「薄い青紫〜薄い赤紫」グラデーションを継承。
  - `#7a49e1`～`#8951f0`。
- 極小サイズでは単色`#824fe3`とする。

## サイズ別パターン
- 256px以上 → 文字入り。`_workspace\icon_with_text.svg`
- 256px未満 → 文字なし。`_workspace\icon_without_text.svg`
- 128px未満？ → 文字なし、図案をシンプルに。`_workspace\icon_small_simple.svg`
- 32px以下 → 文字なし、単色背景、図案を更にシンプルに。`_workspace\icon_smaller.svg`
- SVGマスク → 文字なし、透明背景、図案は黒一色のみ`_workspace\icon_mono.svg`

## 外側余白（Safe Zone）
- 各サイズで全辺に14%の余白
- マスクや角丸で欠けない安全領域を確保

## 文字仕様
- フォント：Rounded Mplus 1c
- 太さ：極太。ウェイト900（可読性重視）
- 字間：やや広め（デフォルトより5〜15%拡大）
- 配置：安全余白内最下部中央
- 文字色：白
- 文字縁取り: `#222222`の薄灰色。線幅は1

## ライセンス表記（フォント）
- フォントファイルは配布しない（画像化のみ）
- README等に素材クレジットとして記載：
  - フォント名
  - 公式配布ページ
  - ライセンス名（SIL Open Font License 1.1）
