# 更新履歴

## v1.6.2 (2025-12-13)

### セキュリティ強化と安定性の向上

- **セキュリティ**: 公開鍵暗号方式による署名認証を導入し、セキュリティを大幅に強化しました。
- **データ管理**: 長期間利用のないデータの自動削除機能を改善しました。
- **バグ修正**: タイムゾーン変更時の時刻調整ロジックを修正しました。
- **バグ修正**: プッシュ通知の購読解除が即座に反映されるように改善しました。
- **ドキュメント**: 更新履歴（CHANGELOG.md）機能を追加し、最新のアプリケーション仕様書（SPECIFICATION.md）などを更新しました。

## v1.6.1 (2025-12-12)

### ドキュメント整備

- **ドキュメント**: 紹介記事（INTRODUCTION.md）を追加しました。
- **UI改善**: ヘッダーにヘルプ（マニュアル）へのリンクを設置しました。

## v1.6.0 (2025-12-08)

### 通知の信頼性向上

- **機能追加**: Web Push通知のリトライ機能を追加しました。通知に失敗した場合、自動的に再送を試みます。
- **機能追加**: 通知失敗が続いたリマインダーを自動的に一時停止し、その旨を表示するようにしました。

## v1.5.0 (2025-12-08)

### 利用制限機能の追加

- **安定性**: サービスの安定運用のために、リマインダーの登録数や文字数などに制限を設けました。

## v1.4.3 (2025-12-07)

### UI/UXの大幅改善

- **UI改善**: リマインダーリストのグルーピング表示（次回通知日、タグ、ステータス別）に対応しました。
- **機能追加**: 次回通知日順での並び替え機能を追加しました。
- **デザイン**: リマインダーカードのデザインを見やすく改善しました。

## v1.4.2 (2025-12-06)

### 操作性の改善とデータ保護

- **操作性**: リマインダー削除中などに誤操作を防ぐロック機能を追加しました。
- **機能改善**: 通知方法（ローカル/プッシュ）切り替え時のデータ同期を改善しました。
- **安全性**: 危険な操作の前に確認メッセージを表示するようにしました。

## v1.4.0 (2025-12-04)

### 機能追加と使いやすさの向上

- **機能追加**: 通知周期の計算に「基準日」を設定できるようになり、より柔軟なスケジュールが可能になりました。
- **UI改善**: 画面右下に通知メッセージ（トースト）を表示するシステムを導入しました。
- **プライバシー**: 長期間利用のないデータを自動削除するプライバシー保護機能を追加しました。

## v1.2.0 (2025-12-03)

### 通知機能の強化

- **信頼性**: Web Push通知の配信基盤を強化し、定期実行の信頼性を向上させました。
- **ドキュメント**: 通知の遅延に関する案内を追加しました。

## v1.1.0 (2025-11-30)

### Web Push通知への対応

- **機能追加**: アプリを閉じていても通知が届く「Web Push通知」に対応しました（設定で有効化が必要）。
- **機能追加**: データのバックアップ（エクスポート）機能を追加しました。

## v1.0.0 (2025-11-30)

### おしらせベル 公開

- **リリース**: URLを添えてリマインド通知するアプリ「おしらせベル」を公開しました。
- **PWA**: ホーム画面へのインストールに対応しています。
